# ACE Quests Block for Moodle

A Moodle block plugin that displays daily learning quests and XP progress from the [ACE (Adaptive Challenge Engine)](https://github.com/letstudygroup/moodle-local_ace) local plugin.

## Features

- Shows daily quests assigned to the student for the current course.
- Displays quest completion status and rewards.
- XP progress bar showing current level and experience points.
- Multi-language support (English and Greek).
- Privacy compliant with full GDPR provider.

## Requirements

- Moodle 4.5 or later
- PHP 8.1 or later
- [local_ace](https://github.com/letstudygroup/moodle-local_ace) plugin installed and configured

## Installation

1. Download the plugin and extract it to `/blocks/ace_quests/` in your Moodle installation.
2. Visit **Site administration > Notifications** to complete the installation.
3. Add the block to any course page via the block drawer.

## Usage

1. Navigate to a course where ACE is enabled.
2. Turn editing on and add the **ACE Quests** block.
3. Students will see their daily quests and XP progress.
4. Completing quests earns XP and contributes to level progression.
